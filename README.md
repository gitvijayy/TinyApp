# TinyAPP

## Goal

Allow users to shorten long URLs much like TinyURL.com and bit.ly do.

Build an HTTP Server that handles requests from the browser (client).

## Dependencies

body-parser

cookie-session

ejs

express

method-override

moment

## Language 

Javascript

HTML

CSS
